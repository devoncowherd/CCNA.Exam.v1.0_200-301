Network Fundamentals: Components

Network Interface Card (NIC)

Definition:

The NIC is a hardware component that enables a computer to connect to a network such a home network and/or the Internet using an Ethernet cable with RJ-45 connector


Types:


Wireless, Wired, USB, and Fiber Optic

Hosts

Definition:

A host is any hardware device that has the capability of permitting access to a network via a user interface, specialized software, network address, protocol stack, or any other means. Some examples include, but are not limited to, computers, personal electronic devices, thin clients, and multi-functional devices.


Types:

Desktops, Laptops, Smartphones, and more!

Switches

Definition:

A device that channels incoming data from any of multiple input ports to the specific output port that will take the data toward its intended [intra-network/LAN] destination.


Types:

Modular, Fixed-Config, Unmanaged, Smart, and Managed

Routers

Definition:

A computer that is a gateway between two networks at OSI layer 3 and that relays and directs data packets through that inter-network[/WAN]. The most common form of router operates on IP packets.


Types:

Wired, Wireless, Edge, Core, and Virtual

Registered Jack (RJ)

Definition:

Any of the series of jacks, described in the Code of Federal Regulations, Title 47, part 68, used to provide interface to the public telephone network - The most common in PC networking being RJ45(Twisted Pair). Note: You can find these jacks on most desktop computers.


Types:


RJ45, RJ11, etc

Firewalls

Definition:

An inter-network connection device that restricts data communication traffic between two connected networks. A firewall may be either an application installed on a general-purpose computer or a dedicated platform (appliance), which forwards or rejects/drops packets on a network. Typically firewalls are used to define zone borders. Firewalls generally have rules restricting which ports are open.


Types:


Packet-filtering, Proxy, NAT, Web App, and Next-Gen Firewalls
